## 🔧 修复策略

### 核心原则

1. **基于核心目标分类**
   - 不根据影响范围，而是根据任务主要目标
   
2. **优先级机制**
   ```
   P1 (内存、资源) > P2 (CI、鸿蒙、音频) > P3 (工具) > P0 (其他)
   ```

3. **一个任务一个分类**
   - 当匹配多个分类时，选择最高优先级的
